Aplication created in JAVA and SPRING BOOT framework. Includes a microservice for managing users and videos. The app is connected to MySQL database. 
Data is downloaded in JSON format.

Technologies:
JAVA
SPRING BOOT
MySQL
JSON
REST API
HTML

//MOVIES
The application allows you to add videos, find videos by title. It is also possible to show movies newer than the entered year and displaying list of all available movies from data base.

//USERS
The application allows you add new user, find user by login, id and email. 
It also allows you to check whether the user with the given login and password is in the database.
